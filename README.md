# Cocktail App

A web app to search and explore cocktails, built with Flask and TheCocktailDB API.

## Features
- Search for cocktails by name
- Browse popular cocktails
- View ingredients and instructions for each cocktail

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `flask run`

## Built With
- Python / Flask
- TheCocktailDB API
